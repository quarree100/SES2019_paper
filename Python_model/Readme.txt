Tested with oemof version v0.3.2

For installing oemof, see:
	https://oemof.readthedocs.io/en/stable/

All parameters are located in:
	data/model_data.xlsx

This parameter set uses the emission scenario "national emission factor - 2030". The other emission scenarios are in "\Scenario_data_emission_factors\emission_factor_scenarios.xlsx" and can be executed by copying the emission factor time series in sheet 'timeseries' column E.
